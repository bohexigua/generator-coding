# generator-coding
generator-coding  javascript 练习算法的脚手架

#### 使用方式

------

`` npm install -g yo generator-coding ``

`` yo coding ``

* 初始化后脚手架，进入创建好的目录下依次执行 `` npm install ``   `` npm run start ``
* 完成后在目录下的 index.js 下开始你的 js 算法编程之旅
* 脚手架目录参考本项目 app/templates/
